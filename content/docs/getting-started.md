Welcome to Shutter, the enterprise Answer Engine Optimization (AEO) and AI Visibility Platform. Shutter audits, tracks, and enhances how Large Language Models like ChatGPT, Claude, Gemini, and Perplexity see and recommend your brand.

## Quick Start Guide

To begin optimizing your presence across neural networks, follow these steps:

### 1. Account Setup

Visit the [Shutter Login Page](/login) and sign in using your Google account. This configures your private console dashboard profile immediately.

### 2. Configure Crawler API Keys

Once signed in, access the **Keys Setup** panel inside the dashboard workspace.
- **OpenAI Key:** Used for GPT-4o real-time diagnostic comparisons.
- **Anthropic Key:** Runs verification probes against Claude-3.5-Sonnet context blocks.
- **Gemini Key:** Grounding evaluation checks for Google AI Overviews.
- **Perplexity Key:** Compares relative share-of-voice indices in Perplexity RAG feeds.

> [!NOTE]
> All API keys are stored securely inside your browser's local storage and are sent in safe API headers to trigger parallel scan probes.

### 3. Run Your First Snapshot Audit

Enter your brand domain name, short description, and competitors in the Scan Input Form. Click **Execute Diagnostic Audit** to launch the observer dome and generate your visibility report.
